translate.py translates the json file values to the other language

to install python and pip
for mac:-
    export PATH="/usr/local/opt/python/libexec/bin:$PATH"   
    export PATH=/usr/local/bin:/usr/local/sbin:$PATH    
    brew install python                                                                      
    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py    
    python3 get-pip.py               

check if python and pip is installed using 
python3 --version
pip3 --version 

this uses googletrans library to install it run following command
pip install googletrans

takes two arguments (do not change the order of arguments)
1 - destination language code
2 - relative path of source language json file (default file name is sourceLanguage)
3 - source language code (only if source language is not in english)
